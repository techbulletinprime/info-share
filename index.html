Java Shastra

Eclipse shortcut

1 CTRL+T or F4 Type Hierarchy: How to check what are the implementation of a interface or a class
2 Ctrl+O To check what are methods present in a class
3 Ctrl+Alt+H Method call hierarchy: check where a particular method is called from
4 Ctrl+Shift+T for finding class even from jar
5 Ctrl + Shift + R for finding any resource (file) including config XML files
6 Ctrl + Shift + o for organize imports
7 Ctrl + / for commenting, uncommenting lines and blocks, see here for live example.
8 Ctrl + Shift + / for commenting, uncommenting lines with block comment
9 Ctrl + Shift + F Select text and press for formatting.
10 Ctrl + F for find, find/replace
11 Ctrl + E Go to other open editors
12 CTRL+Shift+G which searches the workspace for references to the selected method or variable.
13 Alt + ← and Alt + → Hop back and forth through the files you have visited,respectively.
14 Alt + Shift + j to add Javadoc at any place in Java source file.
15 Ctrl + D to delete a line
